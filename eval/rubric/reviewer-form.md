# MathFoundry Human Review Rubric

For each query/system output, score 1-5:

1. Correctness
- Is the mathematical content accurate?

2. Citation adequacy
- Do cited sources support the main claims?

3. Usefulness
- Is the answer useful for actual research workflow?

4. Overclaim risk
- Does the answer overstate unsupported conclusions?

5. Abstention quality (if abstained)
- Was abstention appropriate and helpful?

## Binary checks
- Would you trust this answer as a starting point? (yes/no)
- Did it miss a clearly canonical reference? (yes/no)
